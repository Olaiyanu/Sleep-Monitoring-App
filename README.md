# Sleep Monitoring App

A modern web app to track and improve sleep, featuring a dashboard, sleep tracking, bedtime routines, analytics, goals, achievements, and dark mode.

## Key Features
- **Sleep Tracking**: Log sessions with quality ratings; history saved locally  
- **Dashboard**: Overview of sleep patterns, active goals, and recent achievements  
- **Bedtime Routines**: Customizable routines with progress tracking  
- **Analytics & Insights**: Charts showing trends, sleep quality analysis, and personalized recommendations  
- **Goals & Achievements**: Track goals, unlock badges, and visualize progress  
- **Dark Mode**: Comfortable nighttime viewing with theme toggle  

## Tech Stack
- React + Vite + TypeScript  
- Tailwind CSS  
- Recharts for charts  
- LocalStorage for persistent data  
- Deployed on Vercel  

## Run Locally
```bash
git clone https://github.com/Olaiyanu/Sleep-Monitoring-App.git
cd Sleep-Monitoring-App
npm install
npm run dev
````

Open [http://localhost:3000](http://localhost:3000) in your browser.
